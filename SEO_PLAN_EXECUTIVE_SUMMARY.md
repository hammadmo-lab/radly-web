## Key Improvements by Phase

### Phase 1: Metadata & Schemas (Week 1)
**What:**
- Add Organization schema (company info)
- Add Article schema to 4 content pages
- Add FAQPage schema to instructions
- Add Pricing schema to pricing page
- Add BreadcrumbList schema
- Update 6 meta descriptions for CTR

**Why:** Structured data appears in rich snippets, improves SERP click-through rate

**How:** Edit page.tsx metadata exports, add JSON-LD to layout

**Result:** All pages have complete, optimized metadata

---

### Phase 2: Content & Resources (Weeks 2-3)
**What:**
1. **Blog Hub** (`/blog`) with 5 posts:
   - "Guide to Structured Radiology Reporting" (2,500 words)
   - "Voice Documentation in Radiology" (2,000 words)
   - "AI in Radiology: Safety & Validation" (2,200 words)
   - "HIPAA Compliance for Radiology" (2,000 words)
   - "Radiology Workflow Optimization" (1,800 words)

2. **Feature Pages** (`/features/`):
   - Voice Input
   - Structured Reports
   - Export Formats
   - Templates

3. **Resource Pages**:
   - Use Cases (`/use-cases/` with 5 radiology modalities)
   - Comparisons (vs. dictation, vs. Dragon, vs. generic AI)
   - Integrations (EHR, PACS, etc.)

4. **Internal Linking**: Contextual links from blog to core pages

**Why:** Long-form content targets head and long-tail keywords, builds authority

**How:** Create MDX blog structure, write SEO-optimized posts, add strategic links

**Result:** Targets "radiology AI," "voice documentation," "medical reporting" keywords

---

### Phase 3: Technical SEO (Weeks 2-3)
**What:**
- Sitemap grows from 8 to 40+ URLs
- Each page gets specific canonical tag
- Image alt text added sitewide
- Core Web Vitals dashboard created
- Robots meta tags per-page

**Why:** Better crawlability, prevents duplicate content issues, faster pages

**How:** Update sitemap.ts, add metadata to pages, implement monitoring

**Result:** Google crawls 5x more pages, better crawl efficiency

---

### Phase 4: Ongoing Growth (Month 1+)
**What:**
- 2 blog posts/month (targets emerging keywords)
- 4-6 backlinks/month from industry sites
- Monthly SEO reporting & optimization
- Content updates based on search data

**Why:** Compounding growth, builds authority, sustainable rankings

**How:** Publishing calendar, link building outreach, monthly GSC/GA4 reviews

**Result:** Consistent organic traffic growth 10-20% MoM
